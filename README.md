PROJETO BANCO DE DADOS

Projeto realizado an disciplina de Banco de Dados - IFPB-CG. O objetivo do projeto era criar um modelo relacional de um sistema de vendas, fazendo o Modelo de Entidade e Relacionamento usando o [brModelo](http://www.sis4.com/brmodelo/); Em seguida criamos o modelo Relacional a partir do MER craido anteriormente; Como ultimo passo usamos o XAMPP SERVER para criar o banco de dados utilizando todos os modelos anteriores como base para um banco de dados bem estruturado.
